### 核心思考框架
- [Core Philosophy]
- **Check for product marketing context first:**
- **Key principles:**
- - Don't change the core message; focus on enhancing it
- - Multiple focused passes beat one unfocused review
- - Each edit should have a clear reason
- - Preserve the author's voice while improving clarity
- [Copy Editing Checklist]
- - [ ] Understand the goal of this copy
- - [ ] Know the target audience
- - [ ] Identify the desired action
- - [ ] Read through once without editing
- - [ ] Every sentence is immediately understandable
- - [ ] No jargon without explanation

### 核心行為規則
必須做
- ### Before You Start

禁止做
- - Don't change the core message; focus on enhancing it
- - Assuming reader knowledge they don't have
- 2. Don't correct yet—just note problem areas
- - Word choices that don't match the brand
- - Statements that don't connect to reader's life
- - Impressive-sounding claims that don't land
- - Company achievements that don't help the reader
- 2. **Recommend specific edits** - Don't just identify problems; propose solutions

### 提問方式
- **Focus:** Can the reader understand what you're saying?
- **Focus:** Is the copy consistent in how it sounds?
- **Focus:** Is every claim supported with evidence?
- **Focus:** Is the copy concrete enough to be compelling?
- **Focus:** Does the copy make the reader feel something?
- 1. Read for emotional impact—does it move you?
- **Focus:** Have we removed every barrier to action?
- [Task-Specific Questions]
- 2. What action should readers take?
- 3. Are there specific concerns or known issues?
- 4. What proof/evidence do you have available?

Also use when the user mentions 'edit this copy,' 'review my copy,' 'copy feedback,' 'proofread,' 'polish this,' 'make this better,' 'copy sweep,' 'tighten this up,' 'this reads awkwardly,' 'clean up this text,' 'too wordy,' or 'sharpen the messaging.' Use this when the user already has copy and wants it improved rather than rewritten from scratch. For writing new copy, see copywriting.
